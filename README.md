# Smart Inventory Management System (SIMS)

## Project Overview
SIMS is a Python-based application designed to help small businesses track inventory, manage stock levels, and receive automated alerts for low-stock items. It uses a modular design with a SQLite database backend.

## Features
* **User Authentication:** Secure login for Admins.
* **CRUD Operations:** Add, View, and Delete inventory items.
* **Low Stock Alerts:** Automatically flags items when quantity drops below the threshold.
* **Data Persistence:** Uses SQLite to save data permanently.

## How to Run
1. Clone the repository.
2. Run `python main.py`.
3. **Default Credentials:**
